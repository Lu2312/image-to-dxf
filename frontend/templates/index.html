¿Cómo funciona?

  1. Elige el módulo que necesitas (Cimentación, Planta o Carpintería).
  2. Ingresa los parámetros del proyecto (ejes, dimensiones, materiales).
  3. Haz clic en Preview para ver el SVG en pantalla.
  4. Descarga DXF, PDF o el Catálogo Excel.
  5. El sistema valida automáticamente el cumplimiento de las NTC-RCDF.
