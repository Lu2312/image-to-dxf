2. Descripción del Servicio

Lu CAD Studio proporciona herramientas online gratuitas para:

  • Conversión de imágenes (PNG, JPG, BMP) a formato vectorial DXF
  • Eliminación de fondos y limpieza de imágenes
  • Generación de planos arquitectónicos desde descripciones en texto
  • Acceso a contenido educativo (blog, tutoriales, documentación técnica)
  • API pública para integraciones (sujeta a límites de uso)

El Servicio se proporciona "tal cual" de forma gratuita para uso personal y profesional, con las limitaciones descritas en estos términos.
