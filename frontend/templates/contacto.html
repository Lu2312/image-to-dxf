💬 ¿Cómo podemos ayudarte?

🐛 Reportar un problema técnico

Si encuentras un error, conversión defectuosa o problema en el sitio, escríbenos con:

  • Descripción detallada del problema
  • Pasos para reproducir el error
  • Capturas de pantalla si es posible
  • Navegador y sistema operativo que usas

💡 Sugerencias y mejoras

Valoramos tu feedback. Si tienes ideas para nuevas funcionalidades, mejoras en las herramientas existentes o sugerencias de contenido para el blog, compártelas con nosotros.

🤝 Colaboraciones y alianzas

¿Representas a una institución educativa, fablab, makerspace o empresa? Escríbenos para explorar colaboraciones, integraciones API o planes educativos personalizados.

❓ Consultas sobre uso de las herramientas

Si necesitas ayuda para usar nuestras herramientas, optimizar tus conversiones o entender los parámetros técnicos, estamos aquí para guiarte. También puedes consultar nuestro blog con tutoriales.

📄 Consultas comerciales y API
